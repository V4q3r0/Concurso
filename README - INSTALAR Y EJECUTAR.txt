Para poder ejecutar el programa:

1. Se debe tener instalado XAMP, MAMP, u otro software para la ejecución de phpmyadmin.

2. Se debe subir el script que se encuentra en la carpeta DB a phpmyadmin, ya que está es nuestra base de datos.

Funcionamiento del programa:

1. Para ejecutar el programa es obligatorio ingresar un nombre de usuario, en caso de no hacerlo, no se permitirá ejecutar el programa hasta cumplir dicho proceso.

2. Al pasar el punto anterior, tenemos una interfaz donde nos enseña nuestro username, puntos, nivel de usuario, así mismo nuestra primera pregunta.

3. Se debe seleccionar una de las opciones antes de dar click en el botón para continuar, en caso de no hacerlo no se permitirá la continuación del programa hasta cumplir con está tarea.

4. En caso de no responder correctamente una pregunta, automaticamente se pierde somo direccionados a una pantalla donde nos muestra que hemos perdido.

5. En caso de responder las 5 preguntas correctamente, al final nuestra pantalla final nos dirá que hemos ganado y aparte nos enseñará los puntos acomulados.

6. Como indique en el punto anterior, los puntos se acomulan ((Siempre y cuando no se fallé en ninguna pregunta)) Es decir, al fallar una pregunta los puntos y nivel serán 0 inmediatamente. Al contrario sin importar cuantas ocasiones se ejecute el programa ((Y se use el mismo username)) los puntos se acomularán una y otra vez.
